Modular Multipurpose Calculator

Welcome to the Modular Multipurpose Calculator, a set of calculators designed to assist Pathfinder Second Edition players with optimization. The top ribbon contains our sections; you are currently on the home page. If you wish to perform combat-related calculations, such as average hit chance or damage, use the Combat Calculations page. If you wish to obtain information on how to optimize the class you are playing, use the Class Optimizer page. If you wish to calculate the effects of items, use the Item Effect Calculation page.
